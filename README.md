# O₂ → DONATE₂  
## A Minimal & Beautiful Hover Animation

A single-page micro-interaction that transforms the chemical symbol **O₂ (Oxygen)** into the word **DONATE₂** with a smooth, satisfying animation on hover — all built with pure **HTML + CSS** (no JavaScript!).

Live Demo → https://mohamedilham914.github.io/oxygen/


*(Hover over the O₂ to see the magic!)*

## Features

- 100% pure CSS animation (zero JS)
- Smooth morphing text effect on hover
- Fully responsive & mobile-friendly
- Dark mode ready (looks great on any background)
- Under 2 KB total size — lightning fast
- Perfect for donation pages, environmental campaigns, or creative portfolios

## Tech Stack

- HTML5
- CSS3 (custom properties, transitions, keyframes)
